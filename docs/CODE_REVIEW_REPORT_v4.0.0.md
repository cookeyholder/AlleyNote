## 主要修復成果

### 1. 核心安全性強化 (Security Hardening)
- **HttpOnly Cookie 認證**: JWT Token 已從 `localStorage` 遷移至 `HttpOnly` Cookie，有效防範 XSS 竊取 Token。
- **CSRF 全面啟用**: 驗證並修復了所有變更狀態路由的 CSRF 保護機制。
- **富文本安全性**: 限縮 CKEditor `htmlSupport` 僅允許白名單標籤，並在前端儀表板補齊了漏掉的 `escapeHtml` 跳脫邏輯。

### 2. 架構與路由優化 (Architecture)
- **權限收緊**: 統一將使用者管理、角色管理、系統設定等敏感 API 掛載 `admin` 中介層，防止權限提升漏洞。
- **DI 容器清理**: 移除重複的服務定義，提升啟動效能。

### 3. 測試環境自動化修復 (Test Automation)
- **測試用例更新**: 修復了超過 100 個因建構子參數變更、類別 Final 化、Mock 回傳型別不匹配導致的測試錯誤。
- **資料庫事務修復**: 解決了 SQLite 在測試環境中的巢狀事務衝突問題 (`BEGIN IMMEDIATE`)。
