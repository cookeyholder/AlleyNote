# Attachment — 附件領域

> **Bounded Context**：App\Domains\Attachment
> **適合讀者**：後端開發者

## BC 職責說明

Attachment BC 管理 AlleyNote 中的所有檔案上傳作業：

- **檔案上傳**：接收上傳、儲存至檔案系統、建立資料記錄
- **安全驗證**：MIME type 檢查、檔案大小限制、檔案名稱消毒
- **關聯管理**：附件與公告的關聯
- **清理**：孤立檔案清理
