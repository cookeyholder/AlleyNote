# Statistics — 統計領域

> **Bounded Context**：App\Domains\Statistics
> **適合讀者**：後端開發者、資料分析師

## BC 職責說明

Statistics BC 負責 AlleyNote 的所有數據統計與分析：

- **統計快照**：建立與管理時間點統計快照（Entities\StatisticsSnapshot）
- **分析器模式**：多種類型分析器（Analyzer）獨立計算不同面向統計
- **資料彙總**：聚合原始資料為統計指標
- **查詢與快取**：統計查詢邏輯與結果快取管理
- **匯出**：CSV/JSON 格式匯出

## 依賴方向

```
Statistics ──→ Post（讀取公告資料作為統計來源）
Statistics ──→ Auth（讀取使用者資料作為統計來源）
Statistics ──→ Shared（快取介面、事件系統）
Statistics ←── Application Service（編排與快取管理）
```
