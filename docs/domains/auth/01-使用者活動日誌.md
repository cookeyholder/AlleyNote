# 使用者活動日誌

## 目的

使用者活動日誌系統負責記錄 AlleyNote 中所有重要的使用者操作行為，支援安全稽核、合規需求、異常行為檢測、效能監控與故障排除。

該系統隸屬於 Security Domain，採用 DDD 架構設計，提供強型別安全、高效能索引與 RESTful API 介面。

## 核心元件

### ActivityLoggingService

- **位置**: `app/Domains/Security/Services/ActivityLoggingService.php`
- **責任**: 活動記錄的主要業務邏輯

核心方法：
- `log()` — 通用記錄方法，接受 `CreateActivityLogDTO`
- `logSuccess()` — 記錄成功操作
- `logFailure()` — 記錄失敗操作
- `logSecurityEvent()` — 記錄安全事件
- `logBatch()` — 批次記錄多個活動

支援記錄等級控制（1-5）與動態啟用/停用特定活動類型。

### ActivityLog Entity

- **位置**: `app/Domains/Security/Entities/ActivityLog.php`
- **設計**: 不可變實體，封裝所有活動相關的業務邏輯

枚舉類型：
- **ActivityType**: 21 種活動類型（登入、登出、文章 CRUD、附件操作、安全事件等）
- **ActivityCategory**: 4 個類別（認證、內容、檔案、安全）
- **ActivityStatus**: 4 種狀態（成功、失敗、錯誤、封鎖）

### SuspiciousActivityDetector

- **位置**: `app/Domains/Security/Services/SuspiciousActivityDetector.php`
- **功能**: 失敗率檢測、頻率異常檢測、IP 行為模式分析

### ActivityLogRepository

- **位置**: `app/Domains/Security/Repositories/ActivityLogRepository.php`
- **功能**: 複合索引最佳化查詢、批次操作、分頁查詢、資料清理

## API 端點

| 方法 | 路徑 | 說明 |
|------|------|------|
| POST | `/api/v1/activity-logs` | 記錄單一活動 |
| POST | `/api/v1/activity-logs/batch` | 批次記錄多個活動 |
| GET | `/api/v1/activity-logs` | 查詢活動記錄（支援過濾、分頁） |
| GET | `/api/v1/activity-logs/statistics` | 活動統計分析 |
| GET | `/api/v1/activity-logs/suspicious` | 可疑活動查詢 |

## 資料庫架構

```sql
CREATE TABLE user_activity_logs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    uuid TEXT UNIQUE NOT NULL,
    user_id TEXT,
    session_id TEXT,
    action_type TEXT NOT NULL,
    action_category TEXT NOT NULL,
    target_type TEXT,
    target_id TEXT,
    status TEXT NOT NULL DEFAULT 'success',
    description TEXT,
    metadata TEXT,
    ip_address TEXT,
    user_agent TEXT,
    request_method TEXT,
    request_path TEXT,
    created_at TEXT NOT NULL,
    occurred_at TEXT NOT NULL
);
```

配備 12 個策略性索引（含複合索引）以最佳化查詢效能。

## 與其他服務的整合

- **Auth Domain**: 記錄登入成功/失敗、登出、密碼變更等認證事件
- **Post Domain**: 記錄文章的建立、更新、刪除、檢視操作
- **Attachment Domain**: 記錄附件上傳、下載、刪除、病毒檢測
- **Security Domain**: 記錄存取被拒、IP 封鎖、權限拒絕等安全事件
- **Statistics Domain**: 提供活動統計資料來源，支援圖表輸出

## 使用範例

```php
// 記錄登入成功
$activityLogger->logSuccess(
    actionType: ActivityType::LOGIN_SUCCESS,
    userId: 123,
    targetType: 'user',
    targetId: '123',
    metadata: ['ip_address' => '192.168.1.100']
);

// 批次記錄
$results = $activityLogger->logBatch([$dto1, $dto2]);
```
