# 統計與流量功能

## 範圍

本文件為統計 API、儀表板使用方式與流量追蹤行為的 canonical 功能說明。

## 已實作能力

- 統計總覽與分項端點
- 熱門內容排行
- 瀏覽量時間序列圖表資料
- 登入失敗安全統計
- 透過文章瀏覽流程記錄流量事件

## API 參考入口

請以 [docs/api/README.md](../api/README.md) 作為 API 導覽的唯一入口。

## 維運重點

- 驗證統計功能時，請使用具備統計存取權限的角色。
- 查核查詢參數（`start_date`、`end_date`、`limit`）是否符合控制器限制。
- 若圖表資料異常，需同時檢查資料生成與前端渲染流程。

## 效能期望

- 高基數篩選應走索引支援查詢。
- 高成本聚合查詢應配置適當快取策略。
- 持續監控查詢延遲與錯誤率。
