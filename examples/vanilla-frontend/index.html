DDD 領域驅動設計

採用領域驅動設計原則，實現高內聚、低耦合的企業級架構，確保程式碼的可維護性和擴展性。

JWT 身份驗證

完整的 JWT Token 認證系統，支援 Access Token 和 Refresh Token，提供安全的使用者身份驗證。

RESTful API

標準化的 REST API 設計，支援完整的 CRUD 操作，提供清晰的 API 文件和 Swagger 界面。

SQLite 資料庫

輕量級 SQLite 資料庫，零配置、高效能，支援完整的 SQL 功能和 PDO 操作。

Vite 建構系統

使用 Vite 建構工具，支援熱重載開發、ES6+ 語法和模組化架構，提供極致的開發體驗。

Docker 容器化

完整的 Docker 容器化方案，支援開發、測試、生產多環境部署，提供一致的運行環境。
